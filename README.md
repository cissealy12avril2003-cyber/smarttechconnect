# SmarttechConnect 🚀

**SmarttechConnect** est une plateforme collaborative full-stack permettant la communication et la collaboration en temps réel. Ce projet a été développé pour explorer les technologies de flux de données instantanés et l'architecture des applications web modernes.

---

## 📌 Fonctionnalités Clés
- **Messagerie Instantanée** : Salons de discussion fluides et réactifs grâce aux WebSockets.
- **Tableau Blanc Collaboratif (Whiteboard)** : Espace de dessin partagé en temps réel via Canvas HTML5.
- **Visioconférence P2P** : Intégration du protocole WebRTC pour les flux audio et vidéo en direct.
- **Authentification Sécurisée** : Système complet d'inscription et de connexion avec hachage des mots de passe (Bcrypt).
- **Interface Responsive** : Design moderne conçu pour une utilisation fluide sur mobile et desktop.

## 🛠️ Stack Technique
- **Backend** : Node.js, Express.js
- **Communication Temps Réel** : Socket.io
- **Base de Données** : MySQL
- **Frontend** : HTML5, CSS3, JavaScript (ES6+)
